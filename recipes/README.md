# DiscoClaw Recipes

Integration recipes that teach an AI agent how to extend DiscoClaw.
See `docs/discoclaw-recipe-spec.md` for the format specification.

## Directories

| Directory | Purpose | Shipped? |
|-----------|---------|----------|
| `starters/` | Production-ready recipes to get going fast | Yes |
| `examples/` | Illustrative recipes that teach the format | Yes |
| `community/` | Your own recipes (gitignored) | No |

## Creating a recipe

Use the template at `templates/recipes/integration.discoclaw-recipe.md` or ask the
recipe-generator skill to create one for you.
